To reset the password using Tinker:

Open Tinker:

php artisan tinker


Find the User:
Retrieve the user from the users table using the email:

$admin = \App\Models\User::where('email', 'admin@gmail.com')->first();


Update the Password:
Now, update the password to something new (e.g., newpassword123):

$admin->password = \Illuminate\Support\Facades\Hash::make('newpassword123');
$admin->save();


Check if the Password was Updated:
You can confirm that the password was updated by checking the user record:

dd($admin);


Now, you should be able to log in with the new password (newpassword123)
